# Auro

I build agent-native developer tools and product surfaces: TypeScript/React apps,
orchestration UI, tmux-based agent workflows, and local-first engineering
systems.

My strongest work sits where product engineering, complex state, and AI agent
workflows meet: long-running tasks, approvals, retries, tool calls, session
handoff, and the developer experience around all of it.
